The equipment we use

What we make work

GPS Module

GPS module, small and practical. With an approximate start time of 27 seconds. Able to communicate with a serial connection.

GY-NEO6MV2

Raspbery Pi 3 Model B

- Quad Core 1.2GHz Broadcom BCM2837 64bit CPU
- 1GB RAM
- BCM43438 wireless LAN and Bluetooth Low Energy (BLE) on board
- 40-pin extended GPIO
- 4 USB 2 ports
- 4 Pole stereo output and composite video port
- Full size HDMI
- CSI camera port for connecting a Raspberry Pi camera.

Magnetometer Module

- The L3DG20H gyroscope + LSM303DLHC accelerometer compass + BMP180 barometric/temperature sensors are all on one.
- Since all of them use I2C, you can communicate with all of them using only two wires.

Adafruit 10-DOF IMU
Module LM2596

Voltage regulator

- Vin: 4.5V~35V
- Vout: 1.25V~30V
- Iin: 5A Max
- Iout: 3A nominal. 5A Max
- Minimum voltage difference: 2V.
